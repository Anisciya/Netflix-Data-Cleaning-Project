# Netflix-Data-Cleaning-Project
Data Cleaning and Visualization project using Python, Pandas, and Plotly on the Netflix Titles dataset.

# Netflix Data Cleaning & Visualization Project

## Overview

This project focuses on cleaning, processing, and visualizing the Netflix Titles dataset using Python. The goal is to transform raw data into meaningful insights through data preprocessing and interactive visualizations.

## Objectives

* Handle missing values and duplicate records
* Clean and preprocess the dataset
* Perform exploratory data analysis (EDA)
* Create interactive visualizations using Plotly
* Extract meaningful insights from Netflix content data

## Tools & Technologies

* Python
* Pandas
* NumPy
* Plotly
* Jupyter Notebook

## Dataset

Dataset: Netflix Titles Dataset

The dataset contains information about Netflix Movies and TV Shows, including:

* Title
* Type (Movie/TV Show)
* Director
* Cast
* Country
* Release Year
* Rating
* Genre
* Description

## Data Cleaning Process

The following preprocessing steps were performed:

* Removed duplicate records
* Handled missing values in Director, Cast, Country, and Rating columns
* Cleaned and standardized categorical data
* Exported the cleaned dataset for analysis

## Visualizations

### 1. Movies vs TV Shows Distribution

**Insight:**
Netflix contains significantly more Movies than TV Shows, indicating a strong focus on movie content.

### 2. Netflix Content Growth Over Time

**Insight:**
The number of titles released increased rapidly after 2015, reflecting Netflix's global expansion and increased investment in content production.

### 3. Top Genres on Netflix

**Insight:**
International Movies, Dramas, and Comedies are among the most popular genres available on Netflix.

## Key Findings

* Movies dominate the Netflix catalog.
* Content production and acquisition increased significantly after 2015.
* International content plays a major role in Netflix's library.
* Drama and Comedy are among the most common genres.

## Conclusion

This project demonstrates the complete data analysis workflow, including data cleaning, preprocessing, exploratory data analysis, and visualization. The generated insights provide a better understanding of Netflix's content distribution, growth trends, and genre popularity.
